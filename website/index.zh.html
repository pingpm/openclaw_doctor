工作原理

从崩溃到恢复
只需三步

1

运行安装命令

一条 curl | bash 命令自动安装所有依赖（Node.js、Cloudflare 二进制），并在端口 12222 启动 Doctor 服务，全程无需任何手动配置。

2

终端里出现公网地址

OpenClaw Doctor 使用 Cloudflare 内网穿透，无需路由器端口映射、无需域名备案，约 5 秒内自动生成形如 xxx.trycloudflare.com 的免费公网 URL。

3

把完整链接发给 AI 工具

复制完整链接（含 token），发给任意 AI 工具——Kiro、Cursor、Claude、Trae、Antigravity 等。告诉 AI 你的需求，例如："帮我修复 OpenClaw，远程链接：https://…"。AI 会读取内置操作指南，立即开始工作，无需额外说明。

功能特性

修好一台坏掉的AI智能体
所需的一切

🌐

无需 SSH

只需一个 HTTPS 链接即可访问任意电脑。不需要用户名、密码，不需要开放防火墙端口，不需要配置公钥。

🔐

令牌保护

每次启动自动生成 128 位随机令牌。随时重启更换令牌，旧链接立即失效，安全可控。

🩺

OpenClaw 健康面板

自动运行 openclaw status，将网关状态、活跃 Agent、会话数量、安全警告以可视化方式展示在仪表盘中。

⚡

完整 Shell 权限

执行任意 Shell 命令、读写文件、管理进程——与坐在电脑前操作完全一致，权限与当前系统用户相同。

🤖

AI 工具开箱即用

把完整链接发给 Kiro、Cursor、Claude、Trae 等 AI 工具，AI 读取内置操作指南后立即开始工作，无需额外说明。

🚇

Cloudflare 内网穿透

基于 Cloudflare 免费快速隧道，穿透 NAT、VPN 和防火墙，零配置——不需要账号，不需要域名备案。

📋

操作审计日志

每条命令及其返回结果都会记录时间戳和调用方信息，在仪表盘中按时间顺序查看，随时审计远程操作行为。

🌍

中英文双语界面

仪表盘支持中英文一键切换，默认中文。AI 操作指南使用英文，确保与各类 AI 工具的最佳兼容性。

🖥️

内置 Web 终端

浏览器内置终端，支持命令历史（↑↓）、彩色输出、全屏覆盖模式，无需安装任何额外工具即可手动操作。

API 文档

两个操作
搞定一切

API · application/json · 认证方式：?token=TOKEN 查询参数
# 1. GET 链接 → 返回 AI 操作指南（不要带 User-Agent 请求头）
GET https://xxxx.trycloudflare.com/?token=TOKEN
→ 完整 Markdown 指南：功能说明、命令示例、注意事项

# 2. POST 执行任意 Shell 命令
POST https://xxxx.trycloudflare.com/?token=TOKEN
请求体: { "cmd": "openclaw status" }
响应: { "stdout": "...", "stderr": "...", "code": 0 }

# POST 可选参数
{ "cmd": "...", "cwd": "/路径", "timeout": 60000, "async": true }

# 其他接口（均需 ?token=TOKEN）
GET /status → openclaw status 输出
GET /audit → 完整操作审计日志
GET /info → 会话信息（token、隧道地址、端口）
POST /restart → 重新生成 token，旧链接立即失效
GET /task/:id → 轮询异步任务结果
