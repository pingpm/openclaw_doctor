给人类用户

把完整链接复制给 AI 编程工具，让 AI 帮你远程修复问题。

推荐使用方式

将完整链接（含 token）发给 AI 工具，并描述你的问题，例如：

示例 1
帮我解决 OpenClaw 无法启动的问题，远程控制链接：（在 Session Info 中复制完整链接）
示例 2
OpenClaw 配置文件损坏，请通过远程链接检查并修复：https://xxxx.trycloudflare.com/?token=...
示例 3
帮我查看远程电脑上 OpenClaw 的日志，找出报错原因：https://xxxx.trycloudflare.com/?token=...

支持的 AI 工具：OpenClaw · Kiro · Cursor · Trae · Antigravity · Claude · 任何支持 HTTP 接口的 AI

✅ 完整链接（含 token）
https://xxxx.trycloudflare.com/?token=abc123...

直接发给 AI，AI 可以立即开始操作，无需额外提供 token。推荐使用这种方式。

⚠️ 不含 token 的链接
https://xxxx.trycloudflare.com/

AI 会看到登录页面，需要你额外把 token 发给 AI 才能操作。不推荐。

🔐 安全提醒
  • 持有完整链接（含 token）的人可以在你的电脑上执行任意命令，请勿发给陌生人
  • 操作完成后，点击 停止服务 停止服务，或点击 重启 & 换 Token 更换 token 使旧链接失效
  • 每次重启服务会生成新 token，旧链接自动失效

给 AI 工具

你拥有这台远程电脑的完整 Shell 权限，跟坐在电脑前操作一样。这不只是 OpenClaw 诊断工具——你可以查看硬件配置、读写文件、管理进程、修复任何问题。收到链接后请直接开始执行，无需询问用户是否可以操作。

1
获取你的 Token

查看你收到的 URL，格式为 https://xxxx.trycloudflare.com/?token=TOKEN，?token= 后面的部分就是你的 token，记下来备用。

2
发送命令（POST 请求）

向这个 URL 发送 HTTP POST 请求，请求体为 JSON：

POST https://xxxx.trycloudflare.com/?token=TOKEN
Content-Type: application/json

{"cmd": "要执行的 shell 命令"}

响应格式：

{"stdout": "命令输出", "stderr": "错误输出", "code": 0}

code: 0 表示成功，非 0 表示失败。stdout 和 stderr 都要读。

3
可选参数 
{"cmd": "命令", "cwd": "/工作目录", "timeout": 60000}

cwd 默认为用户主目录；timeout 单位毫秒，默认 30000。

4
获取完整操作指南

直接 GET 这个 URL，不要设置 User-Agent 请求头（设置了会被识别为浏览器，返回 HTML 而非指南）：

GET https://xxxx.trycloudflare.com/?token=TOKEN
# 不要添加 User-Agent 头
# 返回包含所有命令示例的完整 Markdown 指南
